# atm-machine-cli

atm-machine-cli is a cli based interactive atm program created for testing atm experience.

## Features

- Simple, cool and easy to use interface.

## Tech

atm-machine-cli uses a number of open source projects to work properly:

- chalk
- chalk-animation
- figlet
- gradient-string
- inquirer
- nanospinner

## Installation

```sh
npx atm-machine-cli
```
