# cli-atm-machine

cli-atm-machine is a cli based interactive atm program created for testing atm experience.

## Features

- Simple, cool and easy to use interface.

## Tech

cli-atm-machine uses a number of open source projects to work properly:

- chalk
- chalk-animation
- figlet
- gradient-string
- inquirer
- nanospinner

## Installation

```sh
npx cli-atm-machine
```
