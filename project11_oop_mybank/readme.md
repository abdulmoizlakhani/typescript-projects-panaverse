# ninja-bank

ninja-bank is a cli based interactive bank app.

## Features

- Simple, cool and easy to use interface.

## Tech

ninja-bank uses a number of open source projects to work properly:

- chalk
- chalk-animation
- figlet
- gradient-string
- nanospinner

## Installation

```sh
npx ninja-bank
```
