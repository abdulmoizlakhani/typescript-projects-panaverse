# cli-to-doo

cli-to-doo is a cli based interactive todo application created for managing daily basis tasks.

## Features

- Simple, cool and easy to use interface.

## Tech

cli-to-doo uses a number of open source projects to work properly:

- chalk
- chalk-animation
- figlet
- gradient-string
- inquirer
- nanospinner

## Installation

```sh
npx cli-to-doo
```
