# dungeon-master

dungeon-master is a cli based interactive word count & character count application.

## Features

- Simple, cool and easy to use interface.

## Tech

dungeon-master uses a number of open source projects to work properly:

- chalk
- chalk-animation
- figlet
- gradient-string
- inquirer
- nanospinner

## Installation

```sh
npx dungeon-master
```
