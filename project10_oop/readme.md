# oop_example

oop_example is a cli based interactive oop practice app.

## Features

- Simple, cool and easy to use interface.

## Tech

oop_example uses a number of open source projects to work properly:

- chalk
- chalk-animation
- figlet
- gradient-string

## Installation

```sh
npx oop_example
```
